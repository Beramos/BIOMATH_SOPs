## Header structure


### markdown file

```

---
title: "YOUR TITLE"
description: "SHORT DESCRIPTION ON TUTORIAL"
author: "YOUR NAME"
date: YYYY-MM-DD
categories: ["YOUR_CATEGORY"]
tags: ["first_tag", "second_tag", "..."]
---

# your tutorial starts here...
```


### Rmarkdown file


```
---
title: "YOUR TITLE"
description: "SHORT DESCRIPTION ON TUTORIAL"
author: "YOUR NAME"
date: YYYY-MM-DD
categories:
  - YOUR_CATEGORY
tags: ["first_tag", "second_tag", "..."]
output: 
    md_document:
        preserve_yaml: true
---

# your tutorial starts here...
```
