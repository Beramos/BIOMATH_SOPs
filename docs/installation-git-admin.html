git installeren als administrator

Windows

De installatiebestanden zijn beschikbaar via http://git-scm.com/downloads

  1. Voer het installatiebestand uit
  2. Welkom bij de installatie: klik op Next
  3. Aanvaard de licentievoorwaarden door Next te klikken
  4. Installeer git in C:\R\Git
  5. Gebruik de standaard componenten door Next te klikken
  6. Kies use Git from the Windows Command Prompt en klik Next
  7. Kies Checkout Windows-style, commit Unix-style line endingd en klik Next
  8. Kies use Windows' default console window en klik Next
  9. Klik op Finish

Afwijkingen t.o.v. default installatie

  • Installeren naar C:\R\Git i.p.v. C:\Program Files\Git
  • use Git from the Windows Command Prompt
  • use Windows' default console window

Ubuntu

sudo apt-get update
sudo apt-get install git
