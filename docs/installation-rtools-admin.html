Rtools installeren als administrator

Windows

Installatie en upgrade

De installatiebestanden zijn beschikbaar via http://cran.r-project.org/bin/windows/Rtools/

  1. Kies Nederlands als taal voor de installatie en klik volgende.
  2. Klik volgende.
  3. Klik volgende.
  4. Wijzig de installatiemap naar C:\R\Rtools en klik volgende.
  5. Selecteer in de dropdown box Package authoring installation en klik volgende.
  6. Vink Edit the system PATH aan, vink Save versions information to registry aan en klik op volgende.
  7. Klik volgende.
  8. Klik installeren.
  9. Klik Voltooien.

Afwijkingen t.o.v. default installatie

  • Installeren naar C:\R\Rtools i.p.v. C:\Rtools
